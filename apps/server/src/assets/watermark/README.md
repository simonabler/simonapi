# Watermark Assets

Optionale Standard-Logo-Datei für das Watermark-Feature.

- Datei-Name: `default-logo.png`
- Pfad (Entwicklung): `apps/server/src/assets/watermark/default-logo.png`
- Pfad (Build): `dist/apps/server/assets/watermark/default-logo.png`

Wenn vorhanden und bei `mode=logo` kein `logo` hochgeladen wurde, versucht der Service dieses Logo zu verwenden.
