Quick Start

Run locally and test requests instantly: 

nx serve server  # Backend (http://localhost:3000)
nx serve simonapi  # Frontend (http://localhost:4200)

Features

  • QR & Barcodes (PNG/SVG, GS1)
  • Watermarking (logo/text, preview)
  • Locks (access links, swipe‑to‑open)
  • Utilities (echo, IDs, slugify, hash, markdown)

DX & Architecture

NX monorepo, Angular standalone, NestJS, TypeORM, Bootstrap 5. Focus on clear interfaces, good defaults and pragmatic tools.
