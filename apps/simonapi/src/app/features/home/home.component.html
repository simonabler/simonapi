✨ Simon API Hub

APIs, Doku & Tools –
alles an einem Ort

Erzeuge Barcodes & QR, versehe Bilder mit Wasserzeichen, verwalte Locks und nutze nützliche Dev‑Utilities – direkt im Browser.

Zur Dokumentation API ausprobieren

Schnellstart

Starte lokal und teste Requests sofort:

nx serve server  # Backend (http://localhost:3000)
nx serve simonapi  # Frontend (http://localhost:4200)

Features

  • QR & Barcodes (PNG/SVG, GS1)
  • Watermarking (Logo/Text, Preview)
  • Locks (Access‑Links, Swipe‑to‑Open)
  • Utils (Echo, IDs, Slugify, Hash, Markdown)

DX & Architektur

NX Monorepo, Angular Standalone, NestJS, TypeORM, Bootstrap 5. Fokus auf klare Schnittstellen, gute Defaults und pragmatische Tools.
